# Fitness Microservices

A Spring Boot microservices application for managing fitness activities and users.

## Services

- Eureka Server
- Config Server
- API Gateway
- User Service
- Activity Service
- AI Service

## Technologies

- Java
- Spring Boot
- Spring Cloud
- Eureka Discovery
- Spring Cloud Gateway
- Docker
- Maven

## Run

1. Start Docker
2. Run:
   docker-compose up -d
3. Access services through the API Gateway
